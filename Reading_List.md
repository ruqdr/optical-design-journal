# Reading List

A curated list of resources for optical lens design.

## Books

- **Modern Optical Engineering** by Warren J. Smith
- **Fundamentals of Photonics** by Saleh and Teich
- **Introduction to Lens Design** by Joseph M. Geary

## Online Resources

- [Optics Forums](https://www.opticsforums.com)
- [Edmund Optics Technical Library](https://www.edmundoptics.com/knowledge-center/)

## Software

- Zemax OpticStudio
- Code V
- OSLO
- Python libraries: PyOptics, RayOptics
